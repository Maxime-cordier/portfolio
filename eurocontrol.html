Description de ma mission

Au sein de l'organisation européenne Eurocontrol, j’ai participé à un projet visant à centraliser les connaissances internes grâce à un chatbot. L’objectif : faciliter l’accès aux informations grâce à un LLM, tout en garantissant la souveraineté des données de l’entreprise. Lors mon stage, j'ai pu découvrire l'architecture RAG (Retrieval Augmented Generation) permettant d'enrichir les réponses du LLM avec des informations extraites d’une base de données vectorielle, sans nécessiter de réentraînement complet. Je me suis également intéressé au fine-tuning qui consiste à réentraîner un modèle existant avec des données internes d’Eurocontrol. J’ai conçu des scripts Python pour collecter des données internes (documents, rapports) et externes, notamment via du web scraping sur des sites spécialisés comme Skybrary.aero. J’ai ensuite transformé ces données pour les rendre utilisables par les modèles d’IA, en respectant des formats comme JSON ou TXT.

Technologies utilisées

  • Architecture RAG (Retrieval Augmented Generation)
  • Méthodes PEFT (Parameter-Efficient Fine-Tuning)
  • Data Engineering : Python, Beautiful Soup, JSON
  • Modèles : LLaMA-2, h2oGPT, modèles d'embeddings
