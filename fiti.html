Projet FITI

Application mobile de fitness et nutrition

Description du projet

Application mobile développée en React Native pour les adeptes de musculation. Fiti permet la gestion des séances de sport et le suivi nutritionnel. Une version web codée en React permet d'administrer l'application. Fiti a été réalisé dans le cadre d'un projet scolaire à l'Ecole Nationale Supérieure des Sciences Appliquées et de Technologie (ENSSAT) de Lannion. Durant six mois, le suivis du projet s'est déroulé en mode agile avec un jeu de rôle entre professeurs (client) et étudiants. Intégré dans une groupe de 5 étudiants, j'étais principalement chargé de la création et de la gestion de la base de données ainsi que d'une partie du développement fullstack de l'application mobile.

Technologies utilisées

  • React
  • React Native
  • TypeScript
  • Tailwind CSS
  • SupaBase
  • PostgreSQL
  • GitHub

Fonctionnalités principales

  • Création et suivi de programmes d'entraînement
  • Liste de recettes et suivi nutritionnel (Table Ciqual)
  • Statistiques et progression
  • Coach IA (API ChatGPT)
